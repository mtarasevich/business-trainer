# Дизайн-концепция

## Общий стиль
Светлый минимализм, "money green" акцент, много белого пространства, крупная типографика. Ощущение: дорого, спокойно, по-делу. Без градиентов-радуг, без 3D-эффектов, без анимированного фона.

Рефренсы по духу (mental model):
- Stripe (типографика и спокойствие)
- Linear / Vercel (чистая сетка, тонкие границы)
- Лендинги Notion-консультантов (читаемость текста)

## Цветовая палитра

CSS-переменные:

```
--bg:           #FBFCFA   /* почти-белый, уходит в тёплый, лёгкая зеленца */
--bg-alt:       #F1F5EE   /* секции-чередование, чуть зеленее */
--surface:      #FFFFFF   /* карточки */
--ink:          #0F1A14   /* основной текст, почти-чёрный с зелёным подтоном */
--ink-soft:     #46524A   /* подзаголовки, мета */
--ink-mute:     #7A857F   /* подписи, плейсхолдеры */
--line:         #DDE4DA   /* бордеры */

--brand:        #1E7F4F   /* основной зелёный — "цвет денег" */
--brand-deep:   #15623C   /* hover/active */
--brand-soft:   #E2F2E9   /* фон чипов, бэйджи */
--accent:       #C9A24A   /* тёплый акцент (золото) — для цифр, премиум-чувство */

--danger:       #B0463C   /* ошибки формы */
--focus:        #1E7F4F   /* outline focus */
```

Контраст: --ink на --bg = WCAG AAA. --brand на --bg = AA для крупного текста.

## Типографика

Пара шрифтов (Google Fonts):
- **Heading**: "Manrope" 700/800 — современная, гуманная, отлично смотрится в крупных размерах.
- **Body**: "Inter" 400/500/600 — нейтральная, читаемая, идёт в паре.

Альтернатива fallback: system-ui, -apple-system, "Segoe UI", sans-serif.

Размерная шкала (rem, base 16px):
```
--fs-xs:   0.875rem  /* 14 — meta */
--fs-sm:   1rem      /* 16 — body */
--fs-md:   1.125rem  /* 18 — body large, lead */
--fs-lg:   1.5rem    /* 24 — h4 */
--fs-xl:   2rem      /* 32 — h3 */
--fs-2xl:  2.75rem   /* 44 — h2 */
--fs-3xl:  4rem      /* 64 — h1 desktop */
--fs-hero: clamp(2.25rem, 5vw + 1rem, 4.5rem)
```

Line-height: heading 1.1, body 1.6.
Letter-spacing: heading -0.02em, eyebrow uppercase +0.08em.

## Сетка и отступы
- Контейнер: max-width 1200px, padding 24px desktop / 16px mobile.
- Сетка: 12 колонок, gap 24px desktop / 16px mobile.
- Вертикальные ритмы между секциями: 96px desktop / 64px mobile.
- Радиусы: 8px (small), 16px (cards), 999px (chips, buttons).
- Тени:
  - `--shadow-sm`: 0 1px 2px rgba(15,26,20,.04), 0 1px 1px rgba(15,26,20,.03)
  - `--shadow-md`: 0 8px 24px rgba(15,26,20,.06), 0 2px 6px rgba(15,26,20,.04)

## Компоненты

### Header (sticky)
Высота 72px. Фон --bg с blur при скролле, тонкая нижняя линия --line. Логотип слева ("ЕЛ" в кружке --brand-soft с инициалами --brand + текст "Егор Лукшо"). Навигация по центру / справа. Кнопка-CTA primary справа. На мобиле — burger.

### Hero
Layout: 2 колонки 60/40 desktop, стек на мобиле.
- Слева: eyebrow (--brand uppercase, тонкий) → H1 (--fs-hero, переход на новую строку перед "которые") → подзаголовок (--fs-md, --ink-soft) → 2 кнопки → 3 чипа доверия с иконкой-галочкой.
- Справа: фото (main photo.png) в скруглённом прямоугольнике 16px, тень --shadow-md. На фото — overlay-плашка (бэйдж) "20+ лет в продажах" --brand фон.
- Фон секции: --bg, в правом верхнем углу мягкое радиальное пятно --brand-soft (декор).

### Кнопки
- Primary: --brand bg, white text, padding 14/24, radius 999px, hover --brand-deep, focus outline 2px --brand offset 3px.
- Secondary: bordered (--ink, 1.5px), bg transparent, text --ink, hover bg --ink color --bg.

### Карточка цифр (4 в ряд desktop, 2x2 mobile)
- Фон --surface, border 1px --line, radius 16px, padding 28px.
- Большая цифра --fs-2xl, color --brand, font-weight 800.
- Подпись --fs-sm --ink-soft.
- Hover: тень --shadow-md, transform translateY(-2px), 200ms ease.

### Карточка услуги (4 шт)
- Layout: горизонтальный flex-grid 2x2 desktop, стек на мобиле.
- Фон --surface, бордер --line, radius 16px, padding 32px.
- Заголовок H3 --fs-xl, подзаголовок --ink-mute --fs-sm uppercase tracking +0.1em.
- Описание --fs-md.
- Список программ — bullet с иконкой ✓ --brand.
- CTA-ссылка снизу: текст --brand, стрелка → c hover-translate +4px.
- Тонкий цветной верхний бордер 3px --brand при hover (rises feeling).

### Чипы экспертизы
Pill-кнопки, фон --brand-soft, цвет --brand-deep, padding 8/16, --fs-sm, без бордера.
Wrap, gap 8px.

### Блок кейса
Карточка: padding 40px, бордер --line, radius 16px.
Layout: левая колонка 40% — заголовок и итог-цифра огромная (--fs-3xl --brand). Правая 60% — два суб-блока: «Вызов» и «Что сделано» с маленькими H4.

### Образование (timeline)
Вертикальный timeline. Слева — год в круге --brand-soft с цветом --brand-deep. Справа — текст. Соединительная линия --line.

### Видео
YouTube — embed iframe в обёртке aspect-ratio 16/9 с тенью.
Локальное видео — `<video controls poster="...">` с poster (1й кадр ужать до .jpg).

### Галерея
Masonry-подобная сетка 3 колонки desktop / 2 mobile, gap 16px. Lightbox — простейший vanilla (overlay + крупное фото + close + arrows).

### Контакты (блок «Связаться», обновлено 27-Apr-26)
Две карточки вместо трёх дублей:
1. **Email** — карточка с иконкой mail, label «Email», ссылка mailto.
2. **Телефон** — карточка с иконкой phone, label «Телефон», крупная ссылка tel. Под номером — строка «Написать в мессенджер:» + две круглые кнопки 44×44 с inline-SVG логотипами Telegram и Viber. Цвета: Telegram #229ED9 (бренд), Viber #7360F2 (бренд). Hover: подъём 2px + увеличение тени. На мобиле — кнопки в один ряд по центру под номером.

SVG-логотипы — упрощённые версии официальных иконок (бумажный самолётик Telegram, телефон с пузырём Viber). Tooltip / aria-label: «Написать в Telegram», «Написать в Viber».

### Форма
Поля — label сверху --fs-sm --ink-soft, input border 1.5px --line, radius 8px, padding 12/16, focus border --brand, shadow 0 0 0 4px --brand-soft.
Select — кастомный внешне (стрелка SVG), функционал нативный.
Submit — primary, full-width на мобиле.

### Footer
Тонкий, фон --ink, текст --bg/0.7, --fs-sm. Контакты в строку, копирайт ниже.

## Иконография
Inline SVG, stroke 1.5px, цвет currentColor. Набор:
- галочка (для чипов доверия и булитов)
- email / phone / telegram / viber (контакты)
- стрелка → (CTA)
- внешняя ссылка (для подкаста, выставки)

Без иконпаков. Все иконки руками или из Lucide-style.

## Анимации
Минимально:
- fade+up при скролле в видимость секций (intersection observer, 400ms ease-out, distance 16px).
- hover transitions 200ms на ссылках/кнопках.
- Lightbox open/close 200ms.

Никаких parallax, никаких бесконечных background animations.

## Адаптивность
- Mobile: < 768
- Tablet: 768–1024
- Desktop: > 1024
- Wide: > 1440 (контейнер не растёт, фон расширяется)

Hero на мобиле — фото СНИЗУ под текстом (не сверху, чтобы текст был первым).
Шапка на мобиле — burger, drawer выезжает справа.

## Доступность
- Все интерактивные элементы — focus-visible с outline --focus 2px offset 3px.
- Lightbox — trap focus, Esc закрывает, aria-modal.
- Form — label `for=`, `aria-invalid` при ошибке, ошибки рядом с полем.
- alt на каждое фото (описание контекста, не "фото").
- Пропуск-блок «Перейти к содержимому» в начале body.

## Готовые ассеты для сборки
- Логотип: SVG (текстовый, инициалы в круге --brand-soft).
- Иконки: inline SVG в HTML.
- Шрифты: Google Fonts через preconnect + display=swap.
- Изображения: ужать до 1600px max-width, jpg q=80, lazy-load. Hero-фото — отдельная меньше-копия 800px для mobile (srcset).
